# 兩段式六連桿抬球機構
---

本作業將利用Onshape繪製一個能抬起鋼球的兩段六連桿機構，再將球的落點經過模擬測試，繪製簡單形狀的軌道，讓鋼球在機構中成功進行抬起→落至軌道→行經軌道→抬起的循環。

鋼球直徑：8.5mm
